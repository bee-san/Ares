# HackThisSite Basic Missions Writeups

This repository contains my personal writeups and learning notes for the **HackThisSite Basic Missions**.

These challenges helped me understand real-world web security concepts, including authentication flaws, client-side vulnerabilities, and common beginner penetration testing techniques.

This repository documents my learning process, methodology, and security lessons gained from solving each challenge.

---

## About HackThisSite

HackThisSite is a legal cybersecurity training platform designed to help users learn ethical hacking and web security through hands-on challenges.

Platform: https://hackthissite.org

---

## Objectives of this Repository

- Document my cybersecurity learning journey
- Practice penetration testing methodology
- Understand web application vulnerabilities
- Build practical cybersecurity skills
- Create a professional cybersecurity portfolio

---

## Skills Learned


- Website inspection using browser developer tools
- HTML source code analysis
- Identifying authentication flaws
- Understanding client-side vs server-side security
- Cookie inspection and manipulation
- Finding hidden files and directories
- Understanding insecure implementations

---

## Tools Used

- Browser Developer Tools (Inspect Element)
- Console
- View Page Source
- Basic HTML and JavaScript analysis

No automated tools were used. All challenges were solved manually to improve fundamental skills.
